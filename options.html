LMNT character.ai Speech Extension Options

The LMNT character.ai extension is developed solely for entertainment purposes and is not affiliated with character.ai. It is intended to allow folks to experiment with the use of LMNT-produced AI-based synthetic speech to talk with their favorite characters.

For the extension to operate, you'll need an LMNT API key. Please visit the LMNT account page, create a new API key, copy the key and paste it into the field below.
